# HPV_HSsulfation

## Overview
This repository contains MATLAB code used for analysis and modelling related to HPV16 binding and heparan-sulfate sulfation. The code is organized by task into separate folders. Each folder has its own README describing the scripts and functions in more detail.

## Repository structure

### `Kinetics analysis/`
TIRF / tracking analysis scripts for trajectory processing, intensity analysis, cumulative analysis, thresholding, visualization, and quantification.

See:
- `Kintetics analysis/README_TIRF_kinetics.md`

### `Markov kinetics model/`
Scripts and functions for multibond lifetime simulations, survival-curve generation, effective dissociation fitting, and Poisson occupancy calculations.

See:
- `Markov kinetics model/README_Markov_kinetics_model.md`

### `Polymers and diffusion sim/`
Scripts and functions for 2D multivalent diffusion simulations, 3D multi-tether Brownian simulations, and downstream simulated-trajectory analysis.

See:
- `Polymers and diffusion sim/README_Polymers_and_diffusion_sim.md`

### `Simulation SNR/`
Scripts for simulated SNR-related data generation and processing.

See:
- `Simulataion SNR/README_SNR_simulation.md`

### `Tracking 2D/`
Functions for 2D trajectory extraction, cleaning, classification, diffusion analysis, motion correction, and visualization.

See:
- `Tracking 2D/README_Tracking2D.md`

## Notes
- Folder names are preserved as they currently appear in the repository.
- Entry points, scripts, and function call patterns are documented in the folder-specific README files.
